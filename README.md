# WorkHub Booking React App

A professional React web application for booking office spaces, including dedicated desks, conference rooms, and private offices. Built with reusable components and local JSON data for demonstration purposes.

---

## Key Features
- Browse available office spaces, dedicated desks, and conference rooms
- Book desks, rooms, or private offices
- Fully responsive design for desktop and mobile
- Easy-to-use UI built with Bootstrap
- Demo data handled with local JSON files

---

## Tech Stack
 - Frontend Library: React.js

- Languages: JavaScript (ES6+), HTML5, CSS3

- Styling: Bootstrap 5, Responsive Design

- Data: Local JSON (db.json) as mock API (json-server)

- Tools: VS Code, Git & GitHub (for version control), Chrome DevTools

---

## How to Run (for Developers)
git clone https://github.com/tsneethu2016-coder/workhub-booking-react.git
cd workhub-booking-react/js
npm install
# Optional: Start JSON server for mock data
npx json-server --watch data/db.json --port 8000
npm start

The app runs at http://localhost:3000
